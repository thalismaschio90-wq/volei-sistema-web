# VolleyTable Pro Web - Passo 2

## Instalação
Abra o terminal dentro desta pasta e rode:

python -m pip install -r requirements.txt

## Executar
python app.py

Depois abre no navegador:
http://127.0.0.1:5000

## O que este passo entrega
- página inicial
- listagem de competições
- formulário para criar nova competição
- salvamento no dados_volei.json
